Step 1:

Head over to google ads. Go to the campaign you want to plot for.

Step 2:

Go to locations section of campaign.

Step 3:

Make sure Location and Cost columns are shown, then click download, and select Excel .csv

Step 4:

Open up the sheet, and completely remove these selected rows. Do not just delete the data, delete the top two rows entirely.

Step 5:

Delete the data from these rows as well

Step 6:

Done, save changes and file is ready for upload :), all other data validation and input checking is done within the program.
